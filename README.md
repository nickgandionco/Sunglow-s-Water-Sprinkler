# Sunglow-s-Water-Sprinkler
Code repository for automatic water sprinkler project.

MCU used: ATMEGA2560 (Arduino Mega)

Peripherals:
  * 4x20 LCD
  * I2C interface for LCD
  * DS18B20 Temperature sensor
  * DS1307 Real time clock
  * ESP8266 WiFi module
  * 1/2 inch flow meter
  * 1/2 solenoid valve
